Descriptif des objectifs:
- Réalisation d'une liste de Pokémon (Pokédex) représentée par des photos et triée par ordre alphabétique.
- Lorsque l'utilisateur choisi un Pokémon en cliquant dessus, une nouvelle fenêtre s'ouvre contenant les
informations contenant le Pokémon ayant été choisi (id, nom, exp, ...).

Réaliser les fonctions suivantes:
* showPokedex() -> Affichera une liste de tous les Pokémon par ordre alphabétique.
* showPokemon() -> Affichera les informations d'un Pokémon ayant été sélectionné par l'utilisateur dans la liste précédemment définie.

Autres:
Créer le modèle de base pour représenter les Pokémon.
Utiliser Retrofit (client HTTP pour Android) permettant de gérer les requêtes HTTP à envoyer à l'API.
